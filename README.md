# 사물\_숨바꼭질(Object_Hide-and-Seek)

### 설명

ZEP의 메타버스 환경에서 동작하는 스크립트 언어(Scripting Language)를 사용하여 만든 교육형 미니게임 입니다.
<br/>
<br/>
게임 제목은 사물 숨박꼭질(Object_Hide-and-Seek)이고, 2명 이상의 플레이어가 필요한 숨박꼭질 게임입니다. 플레이어의 역할은 술래와 하이더(숨는사람)로 나뉘게 됩니다. 게임을 시작하면 게임시작을 누른 플레이어가 술래가 되고 나머지 플레이어들은 하이더가 됩니다.
<br/>
<br/>
하이더는 술래를 피해 도망가고, 숨어서 최대한 들키지 않고 문제를 찾아내어 푸는게 목표입니다. Q 버튼을 통해 사물(Object)로 변신할 수 있으며, 주변 사물인 척 숨을 수 있습니다. 술래를 피해 문제를 풀면 하이더들이 승리하게 됩니다.
<br/>
<br/>
술래는 하이더들을 찾고 Z(찌르기) 버튼을 사용해 잡는것이 목표입니다. 하이더들이 문제를 풀기전에 사물인척 숨어있는 하이더들을 찾아내서 잡으면 술래가 승리하게 됩니다.
